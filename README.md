# Список дел

Простое React-приложение для управления задачами: добавление, отображение и удаление задач.

## Функциональность

- Добавление новых задач
- Отображение списка всех задач
- Удаление задач
- Счётчик количества задач

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm start

# Сборка для продакшена
npm run build
```
